SoftMind API Endpoints

  1. GET - {{BASE_URL}}/api/v1/auth/
    ➡️ Get a logged in User via Cookie
  2. POST - {{BASE_URL}}/api/v1/auth/login
    ➡️ Authenticate a user and return a token.
  3. POST - {{BASE_URL}}/api/v1/auth/logout
    ➡️ Log out a user and invalidate the token.
  4. GET - {{BASE_URL}}/api/v1/users
    ➡️ Retrieve a list of all users.
  5. POST - {{BASE_URL}}/api/v1/users
    ➡️ Create a User or an Admin
  6. POST - {{BASE_URL}}/api/v1/users/create-super-admin
    ➡️ To Create a Super Admin
  7. GET - {{BASE_URL}}/api/v1/posts
    ➡️ To get all Post
  8. POST - {{BASE_URL}}/api/v1/posts
    ➡️ To Create a Post
  9. GET - {{BASE_URL}}/api/v1/posts/user/:userId
    ➡️ To Get Posts posted by a Specific user
